Der Minecraft Server

⚠️ Achtung!

Der Server ist nur für die Java-Version von Minecraft, ist also nur auf dem PC spielbar.

Auf unserem Minecraft Server, der im Jahr 1869 spielt, werden die Möglichkeiten der industriellen Revolution voll ausgenutzt! Der Server läuft auf der Minecraft Java Version 1.20.1 mit Forge, wodurch wir die Möglichkeit haben, durch Mods das Spielerlebnis noch immersiver zu gestalten.

Durch Mods wie Create kannst du Maschinen bauen, die verschiedenste Aufgaben komplett automatisieren können. Farmer's Delight fügt neue Nahrungsmittel hinzu und Automobility gibt uns die ersten Autos! Um das Erlebnis noch realistischer zu gestalten, benutzen wir Lightman's Currency für ein Wirtschaftssystem und Simple Voice Chat, um mit anderen Spielern auf dem Server reden zu können.

Mods fügen Minecraft neue Funktionen hinzu, wie zum Beispiel neue Blöcke, Items oder Entitäten. Wir haben einige Mods kombiniert, um die industrielle Revolution bestmöglich in Minecraft darzustellen. Hier sind einige der wichtigsten Mods kurz erklärt:

Beitritt

Du möchtest auf dem Server spielen? Super! Wir erklären dir, wie du ganz leicht dem Server beitreten kannst. Denke auch daran, für das beste Erlebnis dem Discord Server beizutreten.

Vorbereitung

  1. Du musst ein Microsoft Konto besitzen, auf dem du die Minecraft: Java Edition oder Minecraft: Java & Bedrock Edition für PC besitzt.
  2. Dein Computer oder Laptop sollte mit mindestens 12 GB Arbeitsspeicher ausgestattet sein.

1. CurseForge

  1. Installiere CurseForge, wenn du es nicht schon installiert hast.
  2. Wähle in CurseForge "Minecraft" aus.
  3. Suche oben nach dem Modpack "RPN - Revolution" oder verwende diesen Link.
  4. Installiere die neueste Version des Modpacks.
  5. Starte das Modpack und betrete den Server mit der IP roleplaynet.de, oder drücke einfach den Join RPN Knopf.

2. Modrinth

  1. Installiere Modrinth, wenn du es nicht schon installiert hast.
  2. Suche oben nach dem Modpack "RPN - Revolution" oder verwende diesen Link.
  3. Installiere die neueste Version des Modpacks.
  4. Starte das Modpack und betrete den Server mit der IP roleplaynet.de, oder drücke einfach den Join RPN Knopf.

3. Andere

  1. Lade dir hier das Modpack als Modrinth .mrpack oder CurseForge .zip herunter.
  2. Importiere es in deinen Launcher. Dein Modloader sollte NeoForge 47.1.106 sein.
  3. Starte das Modpack und betrete den Server mit der IP roleplaynet.de, oder drücke einfach den Join RPN Knopf.

Performance

Da das Modpack über 100 Mods enthält, kann es zu schlechten Framerates oder hohem RAM-Verbrauch kommen. Damit du auch auf etwas schwächeren Geräten spielen kannst, gibt es ein paar Möglichkeiten, um eine bessere Performance zu bekommen

1. RAM-Zuweisung

Deinem Launcher müssen mindestens 8 GB, aber besser 10 GB Arbeitsspeicher zugewiesen sein. Suche dafür nach Optionen wie Maximum Memory Allocation oder ähnlichem, die du auf 8'000 MB bis 10'000 MB setzt.
Solltest du diese nicht finden, kannst du auch den JVM-Parameter -Xmx8G oder -Xmx10G verwenden. Wenn du das Modpack direkt von CurseForge oder Modrinth heruntergeladen hast, ist das schon eingestellt.

2. JVM-Parameter

Der JVM können im Launcher verschiedene Parameter (JVM arguments oder Java arguments) zugewiesen werden. Wenn du eines der folgenden Betriebssysteme benutzt:

  • Windows 10/11
  • Ubuntu
  • Linux Mint Cinnamon
  • Debian GNU/Linux
  • macOS 15 Sequoia
  • macOS 26 Tahoe

dann kannst du die folgenden JVM-Parameter verwenden, um die Performance zu verbessern:

-XX:+UseG1GC
-XX:MaxGCPauseMillis=50
-XX:G1ReservePercent=15
-XX:InitiatingHeapOccupancyPercent=15
-XX:G1HeapRegionSize=16M
-XX:+ParallelRefProcEnabled
-XX:+DisableExplicitGC

Diese funktionieren zwar nicht immer, sollten aber auf den meisten Plattformen die Performance erheblich verbessern.
